Notes

  • The element 'Web adress:' must follow specific validation rules regarding the validity of the URI specs.
  • The element 'Link title:' contains the contents of the markup title tag for suggestions.
  • The element 'Link title:' contains a string of n characters for entry purposes. Its display is spec'd by the user in the area 'Preferences'
  • The elements 'New list:', 'Page:' and 'Add' are only available if the amount of pages is >1.
  • IF the element 'Web adress' is empty and the user activates the button 'Add' THEN the error message 'You have not entered the required information in the mandatory field(s).' is displayed in the status field.
  • IF the element 'link title' is empty THEN the contents of the element will be the Web Address.
  • The element 'Page:' has the current page as its value.
